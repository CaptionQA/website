Leaderboard: Is Your Caption as Useful as the Image Itself?

About CaptionQA

CaptionQA evaluates whether image captions can stand in for images in downstream tasks. Unlike traditional metrics, we measure utility: how well captions preserve the information needed for real-world applications.

Benchmark Stats:
• 4 domains (Natural, Document, E-commerce, Embodied AI)
• 33,027 questions across 657 images
• 50.3 questions per image
• 25 top-level + 69 subcategories

Submit Results

To submit your results to the leaderboard:

  1. Generate captions for our benchmark images using your model
  2. Run our evaluation script (coming soon)
  3. Submit via GitHub PR or email with:
    • Model name and details
    • Results JSON file
    • Paper/tech report link (if available)
    • Code repository link (optional)

Note: Detailed submission instructions will be available with the dataset release.

Evaluation Domains

Natural: Object properties, spatial relations, hallucination (6 categories, 22 subcategories)

Document: Layout, content, structure for automation (6, 15)

E-commerce: Product attributes for recommendations (7, 16)

Embodied AI: Spatial understanding for robotics (6, 16)
