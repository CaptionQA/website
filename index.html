CaptionQA

A Utility-Based Benchmark for Evaluating Image Captions

About CaptionQA

CaptionQA is a benchmark that evaluates whether image captions can stand in for images in downstream tasks. Unlike traditional metrics, CaptionQA measures utility: how well captions preserve the visual information needed for real-world applications.

CaptionQA contains 33,027 multiple-choice questions over 657 images across 4 domains: Natural, Document, E-commerce, and Embodied AI. Each domain has a fine-grained taxonomy (25 top-level and 69 subcategories) defining what information captions should preserve.

How to Submit

Step 1: Download Dev Set

Download the development set (50% of questions + all images) to evaluate your model locally.

Step 2: Generate Captions

Generate captions for all benchmark images using your captioning model. You can use any prompt strategy (Long, Short, Simple, or Taxonomy-Hinted).

Step 3: Submit for Evaluation

Submit your generated captions through our evaluation portal. We will run your captions on the full test set (hidden 50%) and update the leaderboard with both Dev and Test scores.

Step 4: Update Leaderboard

If you have a public paper or technical report, submit a PR to update the leaderboard table with your paper link and model details.

Evaluation Domains

Natural Images (6 categories, 22 subcategories)
Object properties, spatial relations, scene understanding, hallucination detection

Document (6 categories, 15 subcategories)
Layout analysis, text content, table structure, document-specific elements

E-commerce (7 categories, 16 subcategories)
Product attributes, presentation details, UI elements, brand information

Embodied AI (6 categories, 16 subcategories)
Spatial understanding, object affordances, environment perception, task-relevant cues
