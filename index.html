CaptionQA

A Utility-Based Benchmark for Evaluating Image Captions

About CaptionQA

CaptionQA evaluates whether image captions can stand in for images in downstream tasks. We measure utility: how well captions preserve visual information needed for real-world applications.

Benchmark Stats:
• 33,027 questions over 657 images
• 4 domains (Natural, Document, E-commerce, Embodied AI)
• 25 top-level + 69 subcategories
• 50.3 questions per image on average

Submission Workflow

Step 1: Download Dev Set (50% questions + all images)

Step 2: Generate captions with your model

Step 3: Submit captions for full test evaluation

Step 4: PR to update leaderboard with results

Evaluation Domains

Natural - Object properties, spatial relations, scene understanding
Document - Layout, text content, table structure
E-commerce - Product attributes, UI elements
Embodied AI - Spatial understanding, affordances
